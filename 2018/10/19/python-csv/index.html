Python处理csv数据集

首先是读取csv文件，使用自带的库：

import csv
csv_reader = csv.reader(open(‘name.csv’, encoding=’utf-8’))
for row in csv_reader:
print(row)

由于表头是每一列数据的名称，需要去掉，所以要判断数据是否为数字类型：

def is_number(s):
try:
float(s)
return True
except ValueError:
pass

try:
    import unicodedata
    unicodedata.numeric(s)
    return True
except (TypeError, ValueError):
    pass

return False

差不多是这样，最后附一段代码方便我在虚拟机里面用。。。

#encoding:utf-8
import csv

def is_number(s):
try:
float(s)
return True
except ValueError:
pass

try:
    import unicodedata
    unicodedata.numeric(s)
    return True
except (TypeError, ValueError):
    pass

return False

csv_reader = csv.reader(open(‘2circle.csv’, encoding=’utf-8’))
list1=[]
for row in csv_reader:
if is_number(row[1]) and is_number(row[2]):
row[1]=float(row[1])
row[2]=float(row[2])
list1.append([row[1],row[2]])
print(row)

print(list1)
